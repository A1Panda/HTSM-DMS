#### 1. 使用 Docker Compose (推荐)

- **启动**：`docker-compose up -d`
- **更新**：`docker-compose up -d --build` (代码更新后使用此命令即可自动重构镜像并重启)

#### 2. 使用 Docker 命令手动部署

如果您不想使用 docker-compose，可以手动构建和运行：

- **第一步：构建镜像**
  ```bash
  docker build -t htsm-dms .
  ```

- **第二步：启动 MongoDB (可选，如果已有数据库可跳过)**
  ```bash
  docker run -d --name htsm-mongo -p 27017:27017 -v mongo-data:/data/db mongo:6
  ```

- **第三步：启动应用**
  ```bash
  # 链接到上面的 mongo 容器
  docker run -d -p 5000:5000 \
    --name htsm-app \
    --link htsm-mongo:mongo \
    -e MONGODB_URI=mongodb://mongo:27017/htsm-dms \
    htsm-dms
  ```

这样您既可以选择自动化程度高的 Compose 方式，也可以选择更灵活的手动 Docker 命令方式。
