# 产品编码管理系统

## 项目简介

产品编码管理系统是一个用于管理和比对产品编码的Web应用程序。该系统允许用户导入、查看、搜索和比较产品编码数据，帮助企业更有效地管理产品信息。

## 主要功能

- **产品编码导入**：支持导入JSON格式的产品编码数据
- **产品列表展示**：以表格形式展示所有产品信息
- **编码比对**：比较不同产品之间的编码差异
- **统计分析**：显示缺失和多余编码的统计信息
- **扫码功能**：支持扫描条形码快速查找产品
- **YAML配置**：使用YAML格式的配置文件进行系统设置

## 技术栈

- 前端：HTML, CSS, JavaScript
- 后端：Node.js
- 数据存储：JSON文件
- 配置管理：YAML

## 安装指南

1. 克隆仓库到本地
   ```
   git clone [仓库地址]
   ```

2. 安装依赖
   ```
   npm install
   ```

3. 启动服务器
   ```
   node server.js
   ```

4. 在浏览器中访问
   ```
   http://localhost:3000
   ```

## 配置说明

系统配置存储在`public/config.yaml`文件中，可以修改以下配置项：

```yaml
# 界面配置
ui:
  # 默认显示的编码数量
  defaultDisplayCodeCount: 3
  # 提示框最大高度
  tooltipMaxHeight: 300px
  # 提示框网格列数
  tooltipGridColumns: 2

# 数据配置
data:
  # 每页显示的产品数量
  productsPerPage: 12
```

## 使用说明

1. **首页**：显示产品列表和基本信息
2. **产品详情**：点击产品可查看详细信息
3. **编码比对**：选择两个产品进行编码比对
4. **数据导入**：通过导入功能添加新的产品数据
